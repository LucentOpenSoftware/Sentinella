ARGUS Research Samples Directory
================================

This folder is for controlled malware samples used to test and
validate ARGUS detection rules. Files placed here are:

- NEVER automatically scanned by the real-time watcher
- NEVER executed
- Only scanned manually via: sentinella scan runtime/research_samples/
- Used for detection regression testing

WARNING: Files in this directory may contain actual malware.
Do NOT open, execute, or extract any files outside of a controlled
testing environment. Handle with extreme caution.

Use scripts/known-bad-regression.bat for controlled testing.
